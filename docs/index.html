사용 방법

1. ZIP 압축 해제

다운로드한 ZIP을 원하는 폴더에 압축 해제합니다.

2. Npcap 설치

같은 방의 상대방 이모티콘을 보려면 setup 안의 npcap-1.88.exe를 먼저 설치합니다.

3. 실행

setup\AmongEmote.exe를 실행합니다. 런처가 업데이트 확인 후 본체를 시작합니다.

4. 게임 입장

로비 또는 게임 중에 단축키로 이모티콘을 사용합니다.

다운로드
